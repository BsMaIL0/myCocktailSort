# myCocktailSort

Реализация алгоритма «Сортировка перемешиванием» (Cocktail Shaker Sort) на C++ 

## 📋 Обзор

Сортировка перемешиванием — это эффективная вариация пузырьковой сортировки, которая выполняет проходы в обоих направлениях. В отличие от стандартной пузырьковой сортировки, алгоритм чередует:

- **Прямой проход**: «всплытие» наибольшего элемента в конец
- **Обратный проход**: «погружение» наименьшего элемента в начало

Такой двунаправленный подход может быть быстрее стандартной пузырьковой сортировки, особенно для «черепах» (маленьких элементов, находящихся ближе к концу массива).

## ✨ Возможности

- **Шаблонная реализация** — работает с любыми типами данных
- **Поддержка компараторов** — сортировка в любом порядке (возрастание, убывание, пользовательский)
- **Гарантия noexcept** — безопасно использовать в критическом коде
- **Сортировка на месте** — без выделения дополнительной памяти
- **Раннее завершение** — определяет, когда массив уже отсортирован

## 🚀 Быстрый старт

### Базовое использование

```cpp
#include "myCocktailSort.h"

int main() {
    // Сортировка целых чисел
    int numbers[] = {64, 34, 25, 12, 22, 11, 90};
    myCocktailSort(numbers, 7);
    // Результат: {11, 12, 22, 25, 34, 64, 90}
    
    // Сортировка вещественных чисел
    double decimals[] = {3.14, 1.41, 2.71, 1.0};
    myCocktailSort(decimals, 4);
    // Результат: {1.0, 1.41, 2.71, 3.14}
    
    // Сортировка строк
    std::string words[] = {"банан", "яблоко", "вишня"};
    myCocktailSort(words, 3);
    // Результат: {"вишня", "яблоко", "банан"} (лексикографически)
    
    // Пользовательский компаратор (убывание)
    int arr[] = {1, 2, 3, 4, 5};
    myCocktailSort(arr, 5, [](int a, int b) { return a > b; });
    // Результат: {5, 4, 3, 2, 1}
    
    return 0;
}
```

## 📊 Сложность алгоритма

| Случай | Временная сложность | Пространственная сложность |
|--------|---------------------|----------------------------|
| **Лучший** | O(n) | O(1) |
| **Средний** | O(n²) | O(1) |
| **Худший** | O(n²) | O(1) |

**Примечание:** Несмотря на квадратичную сложность в худшем случае, сортировка перемешиванием часто работает быстрее стандартной пузырьковой сортировки, особенно на частично отсортированных данных.

## 🔧 Сборка и тестирование

### Требования

- CMake 3.20 или выше
- Компилятор с поддержкой C++17
- GoogleTest (будет загружен автоматически)

### Инструкция по сборке

```bash
# Клонирование репозитория
git clone <url-вашего-репозитория>
cd myCocktailSort
```

# Конфигурация CMake
```
cmake -B build -DCMAKE_BUILD_TYPE=Release -DBUILD_TESTS=ON
```

# Сборка проекта
```
cmake --build build --config Release
```

# Запуск тестов
```
cd build
ctest --output-on-failure --verbose
```

## 🎯 Когда использовать сортировку перемешиванием

**Используйте сортировку перемешиванием когда:**

- У вас небольшой набор данных (< 1000 элементов)
- Данные частично отсортированы или содержат «черепах» (маленькие элементы в конце)
- Нужна простая сортировка на месте с минимальным кодом
- Вы изучаете алгоритмы сортировки

## 🔍 Визуализация алгоритма

**Пример:** Сортировка `[5, 1, 4, 2, 8]`

```
Проход 1 (прямой): [1, 4, 2, 5, 8] ← 8 «всплывает» в конец
Проход 1 (обратный): [1, 2, 4, 5, 8] ← 1 «погружается» в начало
Проход 2 (прямой): [1, 2, 4, 5, 8] ← Нет обменов, досрочное завершение
```
